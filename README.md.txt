# 📁 Machine Learning & Deep Learning Projects

This repository contains a collection of projects demonstrating practical applications of machine learning and deep learning across diverse domains such as healthcare, finance, computer vision, and customer analytics.

---

## 1. ✈️ Airline Satisfaction

**Objective:**  
Predict customer satisfaction based on in-flight service experience using survey data.

**Highlights:**  
- Feature engineering from categorical and numerical features  
- Model training: Logistic Regression, Random Forest  
- Evaluation using accuracy and ROC-AUC  

---

## 2. 🕵️ Fraud Detection

**Objective:**  
Detect fraudulent financial transactions using machine learning.

**Highlights:**  
- Handled imbalanced data using SMOTE  
- Feature selection and model tuning  
- Models: XGBoost, Random Forest  
- Evaluation: Precision, Recall, F1-score  

---

## 3. ✍️ Handwritten Numbers Recognition (MNIST)

**Objective:**  
Recognize handwritten digits using Convolutional Neural Networks.

**Highlights:**  
- Used the MNIST dataset  
- Built CNN using TensorFlow  
- Achieved >98% accuracy  
- Visualized predictions and training curves  

---

## 4. ❤️ Heart Attack Analysis

**Objective:**  
Predict heart attack risk from patient health data.

**Highlights:**  
- Feature scaling and correlation analysis  
- Models: Logistic Regression, Decision Trees  
- ROC-AUC and confusion matrix evaluation  

---

## 5. 🏘️ Housing Price Prediction

**Objective:**  
Predict housing prices based on property features.

**Highlights:**  
- Data preprocessing and feature engineering  
- Models: Linear Regression, Random Forest  
- Metrics: RMSE, R²  

---

## 6. 🚗 MPG Prediction

**Objective:**  
Predict car fuel efficiency (miles per gallon).

**Highlights:**  
- Regression modeling with continuous/categorical variables  
- Models: Linear Regression, SVR  
- Evaluation: MAE, MSE, R²  

---

## 7. 🤖 Neural Network on CIFAR-10

**Objective:**  
Image classification on CIFAR-10 dataset using CNN.

**Highlights:**  
- Custom CNN architecture and Transfer Learning  
- Data augmentation for improved accuracy  
- Achieved ~75% test accuracy  

---

## 8. 🩺 Pneumonia Detection from X-rays

**Objective:**  
Classify chest X-rays to detect pneumonia.

**Highlights:**  
- CNN and Transfer Learning (ResNet, VGG)  
- Preprocessing: resizing, normalization  
- Metrics: Accuracy, Precision, Recall, F1  

---

## 9. 📉 Customer Churn Prediction

**Objective:**  
Predict whether a customer will churn based on service usage data.

**Highlights:**  
- Categorical encoding, class balancing  
- Models: Logistic Regression, XGBoost, Random Forest  
- ROC-AUC and confusion matrix for evaluation  

---

## 📌 Notes

Each project includes:

Feel free to explore each folder for notebooks, datasets, and insights!
